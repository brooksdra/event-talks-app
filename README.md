# Tech Event Schedule Website

A simple, single-page web application to display the schedule for a one-day technical conference. The front-end is built with standard HTML, CSS, and JavaScript, and the back-end is powered by Node.js and Express.

## Features

- **Dynamic Schedule:** The schedule is dynamically generated by fetching data from a server-side API.
- **Live Search:** Filter talks in real-time by searching for a category.
- **Automatic Timings:** The application automatically calculates the time slots for each talk and break, starting from 10:00 AM.
- **Simple & Clean UI:** A clean and modern user interface for easy readability.

## Technology Stack

- **Backend:**
  - [Node.js](https://nodejs.org/)
  - [Express.js](https://expressjs.com/)
- **Frontend:**
  - HTML
  - CSS
  - JavaScript

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

You need to have [Node.js](https://nodejs.org/en/download/) and [npm](https://www.npmjs.com/get-npm) (which comes with Node.js) installed on your system.

### Installation

1. Clone the repository to your local machine:
   ```sh
   git clone https://github.com/your-username/event-talks-app.git
   ```
2. Navigate to the project directory:
   ```sh
   cd event-talks-app
   ```
3. Install the required npm packages:
   ```sh
   npm install
   ```

### Running the Application

1. Start the server:
   ```sh
   npm start
   ```
2. Open your web browser and navigate to the following URL:
   ```
   http://localhost:3000
   ```
The server will start, and you should see the tech event schedule in your browser.

## API Endpoints

The server provides the following API endpoint to get the talk data:

### Get All Talks

- **URL:** `/api/talks`
- **Method:** `GET`
- **Description:** Retrieves a list of all the talks for the event.
- **Success Response:**
  - **Code:** `200 OK`
  - **Content:** An array of talk objects.
    ```json
    [
        {
            "title": "The Future of Artificial Intelligence",
            "speakers": ["Dr. Evelyn Reed"],
            "category": ["AI", "Machine Learning"],
            "duration": "1 hour",
            "description": "A deep dive into the future of AI and its impact on society."
        },
        // ... more talks
    ]
    ```
